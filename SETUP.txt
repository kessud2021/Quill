Framework Setup Instructions:

1. Copy .env.example to .env
2. Run: composer install
3. Run: npm install
4. Run: npm run build (for production) or npm run dev (for development)
5. Run: php artisan migrate
6. Run: php artisan seed
7. Run: php artisan serve

API Endpoints:
GET    /api/users           - List users
POST   /api/users           - Create user
GET    /api/users/{id}      - Get user
PUT    /api/users/{id}      - Update user
DELETE /api/users/{id}      - Delete user

Auth Routes:
GET    /login               - Login form
POST   /login               - Submit login
GET    /register            - Register form
POST   /register            - Submit registration
GET    /logout              - Logout user

Database Drivers Supported:
- SQLite (default)
- MySQL
- MariaDB
- PostgreSQL

Change database in .env file by setting DB_CONNECTION and credentials.
